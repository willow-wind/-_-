# ums_front

前端采用vue框架开发，使用vuex进行全局权限管理。具有登录注册、角色区分、增删改查和分页检索等功能。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
